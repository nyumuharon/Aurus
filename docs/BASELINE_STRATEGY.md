# Baseline Strategy

## Instrument
XAU/USD

## Timeframes
- execution: 5M
- trend: 1H

## Filters
- trend: EMA50 direction (1H)
- volatility: ATR threshold
- session: London + NY
- spread: must be below threshold

## Entry (Pullback)
Long:
- 1H bullish
- price pulls back to EMA20 (5M)
- bullish confirmation candle

Short:
- inverse

## Stop Loss
- below/above swing
- ATR floor optional

## Take Profit
- fixed 2R

## Position Sizing
- fixed % risk per trade

## Notes
- deterministic
- no ML
- no LLM
